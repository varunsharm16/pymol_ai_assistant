# PyMOL AI Assistant — Electron UI (UI-only)

## Quick start
```bash
npm install
npm run dev
```
Electron will open a window running the React app (Vite dev server).

## Build for production
```bash
npm run build
npm start
```

## Tech
- Electron (main & preload)
- React + TypeScript + Vite
- TailwindCSS
- Zustand (state)
- Dark theme + brand green #7EC636
